##koa2-with-sequelize

>这是一个使用koa2和sequelize实现的demo

### install

```bash
npm install
```

### remind

为了支持koa2，下面组件要注意和1.x不一样
```json
  "koa": "^2.0.0",
  "koa-router": "^7.0.0",
  "koa-views": "^5.0.1"
```

### 技术栈

* koa2
* mongodb
* mysql
* es2015

### run

```bash
babel-node app.js
```
